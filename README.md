
This project is a Python-based system designed to encode and decode football match information using binary representation. The program takes a match code as input, which can be in decimal, binary, or hexadecimal format, and converts it into a structured binary form.

Each part of the binary code represents specific match details, such as the referee type, tournament, stadium, match status, and the two teams playing. The program then decodes these values and displays clear information about the match, including the teams, tournament name, match status, stadium, and referee type.

The system also converts the match code into different number formats (binary, decimal, and hexadecimal), allowing flexibility in how the data is represented and understood.

This project demonstrates the use of binary manipulation, data encoding, and conditional logic in Python, and shows how complex information can be stored and processed efficiently using simple programming techniques.
